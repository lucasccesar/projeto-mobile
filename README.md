# 📚 Bookly Mobile

Aplicação mobile da plataforma **Bookly**, desenvolvida em **Flutter**, que permite aos usuários explorar livros, gerenciar leituras, participar de clubes do livro, favoritar obras e interagir com a comunidade de leitores.

## 🚀 Tecnologias Utilizadas

- Flutter
- Dart
- Material Design 3
- HTTP
- REST API
- Shared Preferences
- Flutter Secure Storage
- Android Studio
- VS Code

## 📱 Funcionalidades

### Usuários

- Cadastro de usuários
- Login e autenticação
- Edição de perfil
- Gerenciamento de informações pessoais

### Livros

- Visualização do catálogo de livros
- Busca de livros
- Consulta de detalhes das obras
- Favoritar livros
- Controle de status de leitura

### Avaliações

- Avaliação de livros
- Visualização de avaliações de outros usuários

### Clube do Livro

- Criação e participação em clubes de leitura
- Gerenciamento de membros
- Troca de mensagens entre participantes
- Discussões sobre livros

### Compras

- Consulta ao histórico de compras
- Visualização dos livros adquiridos

## 🔗 Integração com Backend

A aplicação consome uma API REST desenvolvida em Spring Boot, responsável por:

- Autenticação e autorização de usuários
- Gerenciamento de livros
- Controle de avaliações
- Gerenciamento de favoritos
- Controle de status de leitura
- Administração dos clubes do livro
- Histórico de compras
